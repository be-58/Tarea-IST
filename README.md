# Proyecto PayPhone

Proyecto desarrollado en PHP utilizando la API Sandbox de PayPhone.

## Funciones

- Catálogo de productos
- Carrito de compras
- Pago mediante PayPhone
- Confirmación del pago
- Respuesta del API
- Integración continua mediante Jenkins
- Despliegue automático con Docker

## Tecnologías

- PHP
- PayPhone API
- Docker
- Jenkins
- GitHub
